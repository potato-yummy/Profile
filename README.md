# 🌸 potato-yummy의 첫 번째 웹 프로젝트

> **"HTML과 CSS를 이용해 만든 첫 프로젝트 자기소개 페이지 만들기입니다."** <br>
> 지금까지 배운 내용을 이용해 깔끔한 디자인으로 만들었습니다.

---

## 🔍 작업 중 고민!

### 1. "넓은 화면도 꽉 차 보이게!"
처음에는 화면이 넓어지면 빈 공간이 너무 많아 보여서 고민했습니다. 
여러 디자인을 참고하며 **'나만의 좌우명'**을 넣기로 결정했고, 덕분에 PC 화면에서도 가득 찬 느낌을 줄 수 있었습니다.

### 2. "탭을 했을 때 관련 내용만 보이게!"
가장 어려웠던 건 '강점' 섹션의 탭 클릭 기능이었습니다. 
다양한 검색을 통해 **라디오 버튼(radio)**을 활용하는 방법을 찾아내어 클릭할 때마다 내용이 바뀌는 기능을 완성했습니다.

---

## 🛠 사용한 기술 (Tech)
- **HTML / CSS**: 구조를 잡고 디자인을 입혔습니다.
- **Responsive Design**: PC, 태블릿, 모바일 각 화면 크기에 맞게 보여주도록 디자인했습니다.
